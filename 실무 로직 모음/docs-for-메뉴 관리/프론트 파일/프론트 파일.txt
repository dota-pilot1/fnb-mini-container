1. 레이아웃 컴포넌트

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ components/common/layout/Header.jsx
  - 상단 GNB (Global Navigation Bar)
  - fwMenuState에서 최상위 메뉴(Level 3) 읽어서 렌더링
  - 메뉴 클릭 → handleActive(id) → 좌측 메뉴 업데이트
  - 사용자명 표시, 환경 정보(FS통합결제/FS메뉴) 표시
  - 운영/QA 환경에서는 예제 메뉴 필터링

■ components/common/layout/LeftMenu.jsx
  - 좌측 사이드바 (서브메뉴)
  - 선택된 최상위 메뉴의 하위 항목(Level 4, 5) 표시
  - 접기/펼치기 토글 버튼
  - Level 4: 카테고리 (접기/펼치기 가능)
  - Level 5: 실제 메뉴 항목 (클릭 → 탭 추가 + 페이지 이동)
  - handleActiveLeftMenu(id) → handleAddTab(id)

■ components/common/layout/MdiTab.jsx
  - 열린 페이지 탭 바 (Multi-Document Interface)
  - tabStore에서 탭 배열 읽어서 렌더링
  - 탭 클릭 → handleChangeTab(index)
  - 탭 닫기 → handleCloseTab(index)
  - 홈 버튼, 이전/다음, 전체보기, 전체닫기
  - 최대 12개 탭 제한

■ components/common/layout/ContentsPage.jsx
  - 전체 레이아웃 조합
  - Header + MdiTab + section(LeftMenu + contents)

2. 핵심 훅

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ hooks/common/use-fw-tab.js
  - 메뉴/탭 관련 모든 동작을 관리하는 핵심 훅
  - handleActive(id)         → Header 메뉴 클릭 처리
  - handleActiveLeftMenu(id) → LeftMenu 항목 클릭 처리
  - handleAddTab(id)         → 탭 추가 (최대 12개)
  - handleChangeTab(index)   → 탭 전환
  - handleCloseTab(index)    → 탭 닫기
  - handleClickHome()        → 홈(대시보드) 이동

3. 상태 관리 (Recoil)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ store/atoms/layout/menu-atom.js
  - fwMenuState          : 최상위 메뉴 배열 (Header 렌더링용)
  - fwLeftMenuState      : 현재 좌측 서브메뉴 구조
  - fwLeftMenuOpenAtom   : 사이드바 열림/닫힘 (boolean)
  - fwLeftMenuNameAtom   : 현재 메뉴 이름

■ store/atoms/layout/tabs-atom.js
  - tabStore             : 열린 탭 배열
  - currentTabIndexStore : 현재 활성 탭 인덱스
  - topLevelstore        : 현재 최상위 메뉴 ID

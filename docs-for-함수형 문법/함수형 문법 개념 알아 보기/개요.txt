🛒 쇼핑몰 (추천 ⭐⭐⭐⭐⭐)
장점:
✅ 함수형 연습 많음
상품 필터링 (filter, map)
장바구니 계산 (reduce)
카테고리 트리 (재귀)
검색/정렬 (고차 함수)
✅ 현실적인 비즈니스 로직
가격 계산 (할인, 쿠폰)
재고 관리
주문 처리
✅ 상태 관리 연습
장바구니 (불변 업데이트)
필터 조건 (조합)
함수형 연습 예시:
// 상품 필터링
const filteredProducts = products
  .filter(p => p.category === selectedCategory)
  .filter(p => p.price >= minPrice && p.price <= maxPrice)
  .filter(p => p.inStock)
  .sort((a, b) => a.price - b.price)

// 장바구니 총액
const total = cart.reduce((sum, item) => 
  sum + (item.price * item.quantity), 0
)

// 카테고리 트리
function buildCategoryTree(categories, parentId = null) {
  return categories
    .filter(c => c.parentId === parentId)
    .map(c => ({
      ...c,
      children: buildCategoryTree(categories, c.id)
    }))
}

// 메시지 필터링
const unreadMessages = messages.filter(m => !m.isRead)

// 채팅방 그룹핑
const groupedByDate = messages.reduce((acc, msg) => {
  const date = formatDate(msg.timestamp)
  return {
    ...acc,
    [date]: [...(acc[date] || []), msg]
  }
}, {})
최종 추천: 쇼핑몰 🛒
이유:
함수형 연습 최적
filter, map, reduce 실전 활용
복잡한 계산 로직 (가격, 할인)
트리 구조 (카테고리)
포트폴리오 가치
비즈니스 로직 이해
실무 유사도 높음
단계적 확장 가능
Phase 1: 상품 목록 + 필터링
Phase 2: 장바구니 + 계산
Phase 3: 주문 + 결제
Phase 4: 관리자 (메뉴 관리 응용!)
쇼핑몰 함수형 연습 포인트
// 1. 상품 필터/정렬 체인
products
  .filter(p => p.category === 'electronics')
  .filter(p => p.price < 100000)
  .map(p => ({ ...p, discount: p.price * 0.1 }))
  .sort((a, b) => b.rating - a.rating)

// 2. 장바구니 불변 업데이트
const addToCart = (cart, product) => [
  ...cart,
  { ...product, quantity: 1 }
]

const updateQuantity = (cart, productId, quantity) =>
  cart.map(item =>
    item.id === productId
      ? { ...item, quantity }
      : item
  )

// 3. 가격 계산 (reduce)
const calculateTotal = (items) =>
  items.reduce((total, item) => 
    total + (item.price * item.quantity), 0
  )

// 4. 쿠폰 적용 (함수 합성)
const applyDiscount = (price) => (discount) => 
  price * (1 - discount)

const finalPrice = applyDiscount(10000)(0.2)  // 8000

// 5. 카테고리 트리 (재귀)
const buildTree = (categories, parentId = null) =>
  categories
    .filter(c => c.parentId === parentId)
    .map(c => ({
      ...c,
      children: buildTree(categories, c.id)
    }))
